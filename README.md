# ProjectFlow Dashboard

A task management dashboard built as a capstone project for the **STEMLink SEP Web Module**.

🔗 **Live Demo:** https://project-flow-dashboard.vercel.app/


---

## About

ProjectFlow is a fully responsive task management web application that allows users to create, track, and manage their project tasks in one place.

---

## Features

- Add, delete, and manage tasks
- Filter tasks by status — All, To Do, In Progress, Completed
- Sort tasks by Date, Priority, or Name
- Search tasks by name or description
- Summary stats — Total, In Progress, Completed, Overdue
- Fully responsive — works on mobile, tablet, and desktop
- FAQ accordion section
- Mobile navigation popover

---

## Tech Stack

- HTML
- CSS
- Tailwind CSS
- Vanilla JavaScript
